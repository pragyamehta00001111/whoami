The Path to 20 LPA: DevSecOps Engineer

A highly structured 24-month curriculum. Master these technologies, build public proof-of-work, and target international remote opportunities.

Months 1-3

Phase 1: Linux & Scripting Mastery

  • Linux Internals: File systems, permissions, package management, systemd, SSH.
  • Bash Scripting: Automating system tasks, parsing logs, cronjobs.
  • Python Basics: Data structures, file I/O, writing CLI tools, making API requests.
  • Version Control: Git branch management, rebasing, merge conflict resolution.
Months 4-6

Phase 2: Networking & Cloud Fundamentals

  • Networking: TCP/IP, OSI Model, DNS, DHCP, Subnetting, Firewalls.
  • AWS Core: IAM, EC2, S3, RDS, VPC configurations.
  • Web Servers: Configuring and securing Nginx and Apache.
  • Certification Goal: Pass AWS Certified Cloud Practitioner / SysOps Admin.
Months 7-9

Phase 3: Containerization & Microservices

  • Docker Core: Images, containers, volumes, docker networking.
  • Docker Compose: Orchestrating multi-container local environments.
  • Microservices: Understanding 12-factor app methodology.
  • Registries: Pushing/Pulling from DockerHub and AWS ECR.
Months 10-12

Phase 4: Infrastructure as Code (IaC)

  • Terraform Basics: Providers, state management, resources, data blocks.
  • Terraform Modules: Creating reusable infrastructure code.
  • Configuration Management: Ansible basics (Playbooks, inventories) for server setup.
  • Project: Deploy a full AWS 3-tier architecture entirely using Terraform.
Months 13-15

Phase 5: CI/CD & Automation

  • GitHub Actions / GitLab CI: Writing workflows, managing secrets, triggers.
  • Automated Testing: Integrating unit tests and linting in pipelines.
  • Deployment: Automating releases to staging and production environments.
  • Monitoring Intro: Basic setup of Prometheus and Grafana for metrics.
Months 16-18

Phase 6: DevSecOps & Cloud Security

  • Shift-Left Security: SAST (Static), DAST (Dynamic), and SCA tools integration.
  • Container Security: Scanning images with Trivy or Clair.
  • Secrets Management: Implementing HashiCorp Vault or AWS Secrets Manager.
  • Compliance: Basic understanding of SOC2 / ISO27001 requirements.
Months 19-21

Phase 7: Kubernetes Orchestration

  • K8s Architecture: Pods, Deployments, Services, Ingress, Namespaces.
  • Managed K8s: Deploying clusters via AWS EKS or Azure AKS.
  • Helm: Packaging and deploying applications using Helm charts.
  • GitOps: Implementing ArgoCD or Flux for continuous delivery to Kubernetes.
Months 22-24

Phase 8: The Portfolio & Job Hunt

  • Capstone Project: A massive end-to-end automated infrastructure deployed via IaC, monitored, and secured.
  • Personal Branding: Optimize LinkedIn, polish GitHub READMEs, architecture diagrams.
  • Global Outreach: Target remote-friendly companies in EU/US via Wellfound, WeWorkRemotely, and LinkedIn.
  • Interview Prep: System design, technical Q&A, and mock interviews.
